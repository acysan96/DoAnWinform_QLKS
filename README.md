## Hướng dẫn cài đặt phần mềm Quản lý khách sạn - HotelManager
### Yêu cầu: SQL Server, Windows 10
##### Bước 1: Click đôi chuột vào tệp tin SetupData.bat trong thư mục Setup để cài đặt CSDL.
##### Bước 2: Tại thư mục Setup, mở thư mục Debug, click đôi chuột vào tệp tin setup.exe để cài đặt phần mềm.
